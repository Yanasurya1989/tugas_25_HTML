Mengenal Semantic Elements di HTML5

  • Article

Ketika ingin membuat layout uutuk menampilkan block article-artikel yang ada di website, tag article bisa digunakan dengan element header dan footer, karena tag article memiliki type block, artinya element ini memiliki lebar sesuai browser jika kita tidak menentukan lebarnya.

  • Aside

Element ini bisa digunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan, karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya
